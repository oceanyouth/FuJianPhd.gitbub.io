实数 real number

复数 complex number = 实数real number + 虚数imaginary number

超复数[^1] hypercomplex number

scalar product \ dot product \ inner product \ projection product
vector product \ cross product \ directed area product

1. Grassmann，Hermann Gunther[^2] → 外代数（内积与外积）

  • two vector: exterior product or wedge product or outer product[^3]
    bivector: $ a \wedge b = - b \wedge a $
  • interior product or inner product: $ a \cdot b = b \cdot a $

2. William Rowan Hamilton[^4] → 四元数代数（标量积与矢量积）

  • 四元数 quaternions: william rowan hamilton
    $$ {\color{Red} \omega} +{\color{Green} x} i+{\color{Blue} y} j+{\color{purple} z} k $$

  • real part = scalar part $\Longrightarrow {\color{Red} \omega} $

  • imaginary part = vector part $\Longrightarrow {\color{Green} x} i+{\color{Blue} y} j+{\color{purple} z} k$

  • 两个矢量的标量积（数量积）和矢量积（向量积）：
    $ r_1 = \omega_1 + x_1 i + y_1 j + z_1 k $
    $ r_2 = \omega_2 + x_2 i + y_2 j + z_2 k $
    $ \omega_1 = 0 \space and \space \omega _2 = 0 $
    $
    r_1 r_2 = x_1 x_2 i^2 + x_1 y_2 i j + x_1 z_2 i k + y_1 x_2 j i + y_1 y_2 j^2 + y_1 z_2 j k+ z_1 x_2 k i + z_1 y_2 k j + z_1 z_2 k^2\
    = - (x_1 x_2 + y_1 y_2 + z_1 z_2) + (y_1 z_2 - z_1 y_2) i + (z_1 x_2 - x_1 z_2) j +(x_1 y_2 - y_1 x_2) k\
    = - r_1 \cdot r_2 + r_1 \times r_2
    $

3. Josiah Willard Gibbs[^5] → 点乘和叉乘（点积和叉积）

  • dot product

$ a = \left [ a_1,\cdots ,a_n \right ] $
$ b = \left [ b_1,\cdots ,b_n \right ] $
$ a \cdot b = a_1 b_1 + \cdots + a_n b_n = \sum_{i=1}^{n} a_i b_i = a_i b_i $ (Einstein summation notation)

  • cross product

$ a \times b = a_1 b_1 + \cdots + a_n b_n = \sum_{i=1}^{n} a_i b_i = a_i b_i $

4. Cliffod，William Kingdo[^6] → 几何积（Geometric Product），外代数+四元数代数=几何代数（geometric algebra）

二、四元数物理学[^7]

1. 四元数代数

1.1 四元数的代数运算

1.1.1 四元数的定义

三维空间的实矢量：$\bm{a} = a_1 \bm{i} + a_2 \bm{j} + a_3 \bm{k} $
实数：$ a_0 $
四元数：$ A = a_0 + \bm{a} $
四元数的标部：$ A_S = a_0 $
四元数的矢部：$ A_V = \bm{a} $

1.1.2 代数运算法则

定义：
$ A = a_0 + \bm{a} $
$ B = b_0 + \bm{b} $
$ C = c_0 + \bm{c} $
（1）相等
$ A = B \Leftrightarrow a_n = b_n,\space n = 0,1,2,3 $
（2）加法
$ C = A + B \Leftrightarrow c_n = a_n + b_n,\space n = 0,1,2,3 $
（3）乘法
$ AB = (a_0 + \bm{a})(b_0 + \bm{b})=a_0 b_0 - \bm{a} \cdot \bm{b} + a_0 \bm{b} + b_0 \bm{a} + \bm{a} \times \bm{b} $
$ BA = (b_0 + \bm{b})(a_0 + \bm{a})=a_0 b_0 - \bm{a} \cdot \bm{b} + a_0 \bm{b} + b_0 \bm{a} + \bm{b} \times \bm{a} $
$ \Rightarrow AB \ne BA $
$ a_0 = 0 \space and \space b_0 = 0 $
$ AB = \bm{a} \bm{b} = - \bm{a} \cdot \bm{b} + \bm{a} \times \bm{b} $ [^7]
（4）共轭
（5）模方
（6）逆
（7）除法

1.2 四元数在数中的地位

实数的封闭性——运算结果始终为实数；实数的完备性——除0外任一实数与其它实数乘积可以得到所有实数。

实数→复数→四元数
点→线→面→体

三维矢量不能构成一个代数，因为矢量点乘为标量；叉乘为矢量，但不能遍历整个空间，如不能得到自身。
